vm.$attrs

$attrs 是一个只读属性，它表示父作用域中未被接收的非 prop 特性 (class 和 style 除外) 。

vm.$children

$children 是一个只读属性，它表示当前实例的直接子组件。

vm.$data

$data 是一个属性，它表示 Vue 实例中数据对象的根。

vm.$data-and-data

$data 是一个属性，它表示 Vue 实例中数据对象的根，即 data 选项中定义的数据。

vm.$el-and-el

$el 是一个只读属性，它表示 Vue 实例的根 DOM 元素。

vm.$isServer

$isServer 是一个只读属性，它表示当前 Vue 实例是否运行于服务器。

vm.$listeners

$listeners 是一个对象，它包含了父作用域中（不含.native修饰器的）v-on事件监听器。

vm.$options

$options 是一个只读属性，它包含了创建该实例时传递的选项对象和其他一些选项信息。

vm.$parent

$parent 是一个只读属性，它表示当前实例的父组件实例。

vm.$props

$props 是一个只读属性，它表示 Vue 实例接收 prop 特性的集合。

vm.$props-and-props

$props 是一个属性，它表示 Vue 实例接收 `props` 属性的集合。

vm.$refs

$refs 是一个只读属性，它包含已注册过 ref 特性的所有 DOM 元素和子组件实例。

vm.$root

$root 是一个只读属性，它表示 Vue 实例树的根实例。

vm.$scopedSlots

$scopedSlots 是一个只读属性，它提供了对组件中作用域插槽的访问。

vm.$slots

$slots 是一个只读属性，它提供了对组件中默认插槽与具名插槽内容的访问。
